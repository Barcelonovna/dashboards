# ИИ-дашборды Барселоновны · Barcelonovna AI Dashboards

Коллекция интерактивных дашбордов на открытых и собственных данных.

**Живой сайт:** [https://твой-логин.github.io/название-репо/](https://barcelonovna.github.io/dashboards/)

## Структура

```
.
├── index.html              ← главная с витриной дашбордов
└── population-map/         ← дашборд: население мира до 2100
    └── index.html
```

Каждый дашборд лежит в своей папке и открывается по своему URL:
- Главная: `/`
- Дашборд про население: `/population-map/`

## Как добавить новый дашборд

1. Создать папку с понятным именем (английскими буквами через дефис), например `sales/`
2. Положить туда `index.html` — автономный файл со всем кодом
3. В главном `index.html` найти массив `DASHBOARDS` и добавить новую запись:

```javascript
{
  path: "sales/",
  date: "2026-06",
  badge: { ru: "Продажи", en: "Sales" },
  title: {
    ru: "Продажи по ....",
    en: "sales by ...."
  },
  desc: {
    ru: "Краткое описание дашборда на русском",
    en: "Short description in English"
  },
  meta: {
    ru: ["5 маркетплейсов", "2024–2026", "Внутренние данные"],
    en: ["5 marketplaces", "2024–2026", "Internal data"]
  }
}
```

Карточки на главной автоматически сортируются по дате убыванием.

## Стек

Чистый HTML/CSS/JS, без сборки. D3.js и topojson подключаются с CDN. 
Один файл = один дашборд, открывается локально и с любого статического хостинга.

## Хостинг

GitHub Pages. Любой коммит в `main` автоматически публикуется через 1–2 минуты.

## Дизайн-система

Стайл-гайд «Барселоновны» применяется ко всем дашбордам коллекции:
тёплый off-white фон, бежевые плашки, единый акцентный синий, шкала роста/падения от тёмно-красного через бежевый ноль к тёмно-зелёному.
